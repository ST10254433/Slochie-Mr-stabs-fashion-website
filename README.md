# Slochie-Mr-stabs-fashion-website
Slochie Mr Stabs is a responsive men’s fashion brand website built using HTML, CSS, and JavaScript, focusing on modern UI design and interactive user experience.

## Overview

Slochie Mr Stabs is a responsive men’s fashion brand website designed to showcase a modern clothing concept. The project focuses on clean UI design, responsiveness, and basic front-end interactivity.


This project was built to strengthen front-end development skills using core web technologies.


## Features

- Responsive design for mobile, tablet, and desktop

- Clean and modern user interface

- Structured product/fashion layout

- Interactive elements using JavaScript

- Smooth navigation and user-friendly experience


## Technologies Used

- HTML5 - page structure

- CSS3 - styling and responsive layout

- JavaScript - interactivity and dynamic behavior


## Purpose of the Project

The purpose of this project is to demonstrate front-end development skills by building a real-world style fashion brand website. It focuses on UI/UX principles, responsiveness, and clean code structure.


## Responsive Design

The website is fully responsive and adapts to different screen sizes including:

- Mobile phones

- Tablets

- Desktop screens
